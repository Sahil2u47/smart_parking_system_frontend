# 🚀 Smart Parking System

## 📌 Project Overview
The Smart Parking System is a full-stack web application designed to simplify parking management. It allows users to check real-time parking slot availability, book slots, and manage reservations efficiently. The system also provides an admin panel for monitoring and managing parking operations.

---

## 🎯 Objectives
- Reduce time spent in finding parking
- Provide real-time slot availability
- Enable online booking system
- Improve parking management efficiency

---

## 🛠 Tech Stack

### 🔹 Backend
- Java
- Spring Boot
- Spring Data JPA
- REST APIs

### 🔹 Frontend
- React.js
- Axios
- HTML, CSS, JS

### 🔹 Database
- MySQL

---

## ✨ Key Features

### 👤 User Features
- View available parking slots in real-time
- Book and cancel parking slots
- User authentication (Login/Signup)

### 🛠 Admin Features
- Manage parking slots
- View all bookings
- Update slot status (Available/Occupied)

---

## 🧱 System Architecture
- Frontend (React) communicates with backend using REST APIs
- Backend (Spring Boot) handles business logic
- Database (MySQL) stores user and parking data

---

## 🔄 API Flow (Example)
1. User sends request from frontend
2. Backend controller receives request
3. Service layer processes logic
4. Repository interacts with database
5. Response sent back to frontend

---

## ▶️ How to Run the Project

### 🔹 Backend Setup
1. Clone the repository
2. Open in IDE (IntelliJ/Eclipse)
3. Configure MySQL database
4. Run Spring Boot application

### 🔹 Frontend Setup
1. Navigate to frontend folder
2. Run `npm install`
3. Run `npm run dev`
